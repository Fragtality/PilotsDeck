Run Installer PilotsDeck has external Software Dependencies in order to run at all and do something useful.
It has its own Installer which checks and install all these Dependencies for you - just click 'Run Installer' and the current Release will be downloaded and executed.
Your StreamDeck Software will be restarted during Installation.

NOTE: The Installer might be blocked by Windows Defender or other Anti-Virus Software. If the Installer can not be executed or you want to download it manually, you can get the Installer from GitHub (under Releases).

Following Dependencies will be checked and installed by the Installer:
- Microsoft .NET 8.0 Desktop Runtime x64
- StreamDeck Software (if below minimum)
- If MSFS installed: MobiFlight Event Module
- If FSX/P3D/MSFS installed: FSUIPC
- Optional: vJoy System Driver (Brunner Fork)
